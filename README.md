# vinceladotcom

Personal single-page site built without JavaScript frameworks.

## Requirements

- Node.js 20+
- npm 10+

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Parcel starts a local dev server with fast rebuilds.

## Production Build

```bash
npm run build
```

The production-ready output is written to `dist/` with bundled and minified assets.

## Preview Build

```bash
npm run preview
```
